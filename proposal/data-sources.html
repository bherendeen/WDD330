Site Plan - Data Sources / Features

Building the site

HTML, SASS (CSS), and Javascript will be used to create the site. Responsive design will also be included.

API

The Solar System OpenData API will be the source of obtaining the information needed for the planets and other various aspects.

The possible use of API's from NASA.gov could be used.

Local Storage

Local storage will be used t in the development of light and dark modes. This preference will be saved in local storage to remember the end-users wish to view the site in light or dark modes regardless of whether the user refreshes the browsers and returns to the site later. Colors will automatically adjust for preferences used.

CSS Animations and Transitions

CSS animations and transitions will be used throughout the site to increase the visual aspect and keep children engaged, wanting to learn more, and exploring the planets around them.

Javascript Events

Events will help with the interactions between the children and the site. They'll be able to select the planets they would like to learn more about and interact with a built-in calculator. The calculator will be a feature to check out how much they would weigh if they were on a particular planet.

Javascript Animations

Some animations will be achieved through Javascript Animations using GSAP.js. This is a desire to push me since I just came across it and would like to incorporate it into the site.
